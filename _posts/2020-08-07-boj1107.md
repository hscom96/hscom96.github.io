#느낀점
어느 숫자에 특정 숫자배열 요소들이 포함되어있나 검사하는 로직에서
복잡도를 높게 로직을 구현했다.

```java
public static boolean isContain(int[] err, int ch) {
        while (ch > 0) {
            for (int er : err) {
                if (er == ch % 10) {
                    return true;
                }
            }
            ch /= 10;
        }
        return false;
    }
```

이런 방식보다 아래와 같이하면 o(n2)에서 o(n)으로 복잡도를 줄일 수 있다.
